Files to Create

Create these files exactly in this order

bash — Create Phase 6 foldersterminal
mkdir -p .github/workflows infra/terraform/envs/prod-aws monitoring/grafana k6 security
yaml — GitHub Actions deploy.github/workflows/deploy.yml
name: Deploy Feedback Intelligence Platform
on:
  push:
    branches: [main]
jobs:
  build-test-deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: hashicorp/setup-terraform@v3
      - name: Validate Terraform
        run: terraform -chdir=infra/terraform/envs/prod-aws validate
      - name: Python compile check
        run: python -m compileall apps services shared || true
      - name: Build images
        run: echo "Build and push images to ECR here"
      - name: Deploy Helm
        run: echo "helm upgrade --install feedback-platform infra/helm/feedback-platform -f values-prod.yaml"
terraform — Prod providerinfra/terraform/envs/prod-aws/provider.tf
provider "aws" { region = var.aws_region }
terraform — Prod variablesinfra/terraform/envs/prod-aws/variables.tf
variable "aws_region" { type = string default = "us-west-2" }
variable "environment" { type = string default = "prod" }
javascript — k6 smoke testk6/load_test.js
import http from 'k6/http';
import { check, sleep } from 'k6';
export const options = { vus: 10, duration: '1m' };
export default function () {
  const res = http.get('http://localhost:8000/health');
  check(res, { 'status is 200': r => r.status === 200 });
  sleep(1);
}
yaml — Network policysecurity/network-policy.yaml
apiVersion: networking.k8s.io/v1
kind: NetworkPolicy
metadata:
  name: feedback-default-deny
  namespace: feedback
spec:
  podSelector: {}
  policyTypes: [Ingress, Egress]

Troubleshooting

Common errors and direct fixes

ErrorLikely CauseFix
Terraform initialized in an empty directoryTerraform files were not created first.Create every file under infra/terraform/envs/local-floci/ before running make bootstrap.
Command not foundMissing local tool.Run install commands and then make doctor.
Import errorPackage path mismatch.Use underscore package paths such as apps/agent_service for Python imports.
